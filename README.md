# 🔗 URL Management Application

A secure and user-friendly Spring Boot application that allows users to register, log in, and manage their personal list of URLs. The application uses JWT (JSON Web Tokens) for authentication and authorization, and securely stores user credentials using password encryption.

---

## 🚀 Features

- ✅ User Registration & Login (with JWT token)
- ✅ Password Encryption using BCrypt
- ✅ Add / View / Update / Delete URLs
- ✅ Users can only access their own URLs
- ✅ Profile Update and Delete Account
- ✅ Secure REST APIs using Spring Security

---

## 🛠️ Tech Stack

- **Backend**: Java, Spring Boot
- **Security**: Spring Security, JWT
- **Database**: PostgreSQL / MySQL / H2 (choose your DB)
- **Build Tool**: Maven
- **Testing Tool**: Postman
